# Aspect-Based-Sentiment-Analysis-on-Product-Reviews
**Built a BERT-based ABSA system to extract product features and predict sentiment for detected aspects, achieving 90% accuracy, by implementing an adaptive approach that combines aspect level and overall sentiment analysis with an interactive Streamlit dashboard for real-time insights.**

This project presents a BERT-based Aspect-Based Sentiment Analysis (ABSA) system designed to analyze product reviews at both aspect and overall sentiment levels. The system identifies product features mentioned in reviews and classifies their sentiments as Positive, Negative, or Neutral using deep learning and Natural Language Processing (NLP) techniques.

An adaptive prediction approach is implemented by combining an aspect-level sentiment model with an overall sentiment model to ensure reliable predictions even when no explicit aspects are detected. The application also includes an interactive Streamlit dashboard for review submission, analytics visualization, sentiment distribution, aspect-wise insights, and review history management.

The project demonstrates the practical application of transformer-based models for extracting meaningful insights from unstructured and noisy product reviews, helping businesses and users better understand customer opinions and product performance.
